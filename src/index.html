DOCTYPE - Tells the browser this is an HTML5 document.

html - The main container element for all HTML code.

body - Contains all visible content shown to the user.

h1 - Defines a top level heading.

article - Defines a section of independent, self-contained content.

h2 - Defines a second level heading within an article.

p - Creates a paragraph of text.

b - b is used to bold the text
